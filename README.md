## Учебный проект CRM система колледжа

#### для начала надо скачать код с гитхаба
##### откройте папку в VSCODE и откройте терминал
##### в терминале пишем:
###### python -m venv venv
###### .\venv\Scripts\activate
###### pip install -r requirements.txt
###### python manage.py makemigrations
###### python manage.py migrate
###### python manage.py createsuperuser
###### python manage.py runserver
###### открываем браузер
###### пишем в адресной строке http://127.0.0.1:8000 
###### админка пишем в адресной строке http://127.0.0.1:8000/admin/
